## 🎯 قابلیت‌های اصلی

### 1. مدیریت ادمین‌ها
- ✅ ایجاد ادمین جدید
- ✅ تعیین سطح دسترسی (5 سطح)
- ✅ تغییر رمز عبور
- ✅ فعال/غیرفعال کردن
- ✅ حذف ادمین
- ✅ مشاهده لاگ فعالیت‌ها

**مسیر**: `/admin/management`

---

### 2. ویژه کردن آگهی (مهم! 💰)
- ✅ ویژه کردن دستی توسط ادمین
- ✅ تعیین مدت زمان (1-365 روز)
- ✅ نمایش در بالای لیست
- ✅ علامت ویژه روی آگهی
- ✅ ثبت در جدول featured_listings

**API**: `POST /api/admin/listings/:id/toggle-featured`

**نحوه استفاده**:
1. به پنل ادمین بروید
2. آگهی مورد نظر را انتخاب کنید
3. روی "ویژه کردن" کلیک کنید
4. مدت زمان را انتخاب کنید (مثلاً 30 روز)
5. تایید کنید

---

### 3. تایید/رد آگهی‌ها
- ✅ تایید آگهی: `POST /api/admin/listings/:id/approve`
- ✅ رد آگهی: `POST /api/admin/listings/:id/reject`
- ✅ ویرایش آگهی: `PUT /api/admin/listings/:id`
- ✅ حذف آگهی: `DELETE /api/admin/listings/:id`

## 🚀 شروع کار

### مرحله 1: ورود به پنل
```
1. به http://localhost:8080/admin/login بروید
2. نام کاربری: admin
3. رمز عبور: admin123456
4. کلیک روی "ورود"
```

### مرحله 2: تغییر رمز عبور
```
1. به /admin/management بروید
2. روی آیکون کلید کنار نام خود کلیک کنید
3. رمز عبور جدید را وارد کنید
4. ذخیره کنید
```

### مرحله 3: ایجاد ادمین‌های جدید
```
1. در /admin/management روی "ایجاد ادمین" کلیک کنید
2. اطلاعات را وارد کنید
3. نقش مناسب را انتخاب کنید
4. ذخیره کنید
```

### مرحله 4: تست ویژه کردن آگهی
```
1. به /admin بروید
2. یک آگهی را انتخاب کنید
3. روی "ویژه کردن" کلیک کنید
4. مدت زمان را انتخاب کنید (مثلاً 30 روز)
5. تایید کنید
6. به صفحه اصلی بروید و آگهی ویژه را ببینید
```

## 📝 API های مهم

### ورود ادمین
```bash
POST /api/auth/admin/login
{
  "username": "admin",
  "password": "admin123456"
}
```

### لیست ادمین‌ها
```bash
GET /api/admin/management/list
Authorization: Bearer {TOKEN}
```

### ویژه کردن آگهی
```bash
POST /api/admin/listings/:id/toggle-featured
Authorization: Bearer {TOKEN}
{
  "duration_days": 30
}
```

### لاگ فعالیت‌ها
```bash
GET /api/admin/management/activity-log
Authorization: Bearer {TOKEN}
```
